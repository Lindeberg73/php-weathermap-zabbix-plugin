PHP Weathermap Zabbix plugin

PHP Weathermap plugin integrating Zabbix data (through Zabbix API)

Download this project as a .zip file Download this project as a tar.gz file

This plugin integrates the Zabbix monitoring tool into PHP Weathermap, using Zabbix JSON API. It tends to provide the same functionnalities as the Cacti plugin :

  • Bandwith data
  • Integration of bandwith and host graph
  • Easy configuration using the Zabbix names, accessed using Zabbix web API

Requirements

The plugins requires:

  • Zabbix >= 2.0 (but it is probably easy to adapt it for older releases)
  • PHP Network Weathermap >= 0.97b (but it is easy to adapt it for older releases)
